# 🏇 競馬予想AI システム

DESスコア（A〜Dの4軸評価）による自動競馬予想・収支管理システム。

## 📊 フロントエンド
- [競馬予想AI ダッシュボード](https://aipapa247272.github.io/keiba-auto-gist-updater/)

## 🗂️ ファイル構成

| ファイル | 説明 |
|---------|------|
| `index.html` | メインUI（予想・結果・統計・バックテスト） |
| `scripts/select_predictions.py` | レース選定・買い目生成（v12） |
| `scripts/fetch_race_results.py` | レース結果取得・仮想収支計算 |
| `scripts/daily_select.py` | 日次バッチ実行スクリプト |
| `docs/統合ルール.md` | 運用ルール一次ソース（DES7.x） |
| `final_predictions_YYYYMMDD.json` | 日別予想データ |
| `race_results_YYYYMMDD.json` | 日別結果データ |
| `latest_predictions.json` | 最新予想（フロントエンド用） |

## 🔧 主要ロジック（v12）

### 93%法則 v2（2026-03-05〜）
三連複フォーメーションのcol2に**人気1〜3位を強制追加**。  
統計的に93%のレースで人気1〜3位が3着以内に入ることを活用。

### 若馬限定戦スキップ（2026-03-05〜）
`新馬` / `未勝利` / `2歳` / `3歳` を含むレースを自動スキップ。  
データ不足・能力不安定による予測精度低下を回避。

### 断層分析（v12コア）
DESスコアの「断層（ギャップ）」を検出し、  
レースタイプ（2断層・1断層・フラット）を自動判定。  
フラット（混戦）は自動スキップで投資効率を向上。

## 📈 統計・バックテスト機能
フロントエンドの **統計タブ** より：
- 期間選択（7日 / 14日 / 30日 / 全期間）
- 波乱度フィルター（全体 / 低 / 中）
- KPI6項目（的中率・回収率・損益など）
- 累計収支推移グラフ（SVG）
- 日別バックテスト結果テーブル
